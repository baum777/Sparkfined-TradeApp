# Repo Audit

Use this skill when:
- the user asks for a codebase review
- readiness needs assessment
- architecture quality needs grading
- missing safeguards should be found

Audit dimensions:
- architecture coherence
- module boundaries
- runtime safety
- observability
- testability
- deployment readiness
- governance/compliance surface
- documentation quality

Output format:
- Executive Summary
- Strengths
- Weaknesses
- Critical Risks
- Production Readiness Score
- Immediate Fixes
- Step-by-step transformation path
