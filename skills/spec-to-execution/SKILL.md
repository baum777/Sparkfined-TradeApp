# Spec to Execution

Use this skill when:
- a spec already exists
- a readiness plan exists
- phases must become concrete tasks
- the user asks for next-step prompts or implementation sequencing

Your job:
- extract hard requirements
- preserve prior decisions
- convert scope into bounded tasks
- define exact next move
- include test/verification actions

Output format:
- Scope slice selected
- Why this slice is next
- Implementation tasks
- Files likely affected
- Verification
- Done criteria
