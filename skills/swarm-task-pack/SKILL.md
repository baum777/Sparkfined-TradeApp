# Swarm Task Pack

Use when:
- the user wants work split across multiple agents
- planner / implementer / reviewer / QA roles should be separated
- a bundle for external agents should be generated

Produce:
- role assignment
- task boundaries
- handoff order
- acceptance criteria
- review loop
- merge/reconciliation notes
